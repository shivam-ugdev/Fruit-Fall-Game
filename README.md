# Fruit Fall
Fruit Fall is a 2D arcade-style game built with Unity and C#.  
Players catch falling fruits to score points while avoiding misses.
## Features
- Endless arcade gameplay
- Increasing difficulty
- Score and high-score system
- Sound effects and background music
- Main menu and game over screen
